# Credit Risk Prediction using Machine Learning

## Problem
Predict whether a borrower will default on a loan.

## Dataset
Kaggle Credit Risk Dataset (32,581 records)

## Results

Best Model: Random Forest

Accuracy: 0.93  
ROC-AUC: 0.93  

Key Predictors:
- loan_percent_income
- person_income
- loan_int_rate

## Tools
Python  
Scikit-learn  
Pandas  
Seaborn  
Matplotlib
